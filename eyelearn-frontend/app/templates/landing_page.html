EyeLearn

EyeLearn is web application that allows students to learn the basics of a new language using Machine Learning techniques.

Find Out More

Doodle Classification

EyeLearn enables a student to learn vocabulary by drawing on the screen.

Finding Every Day Objects

EyeLearn allows you to find every day objects using real-time image classification.

Pronunciation

EyeLearn can improve your foreign accent and pronunciation using speech recognition.

Translate Speech

Using EyeLearn tools are available to translate your speech into the language you are learning.

Translate the World around You

EyeLearn enables a user to translate the world around them using just their smart device and Image Classification techniques.

Phrase Translation & Tagging

In addition to speech and object translation, a student can translate their own textual input and have it Part-Of-Speech Tagged, showing the true nature of each word.

Analytics

EyeLearn has ana analytics utility which enables teachers to monitor class progression. Students can also monitor their own performance and see how they rank in the class.

How To Get Involved

EyeLearn is a device-agnostic application and so it can be accessed via any type of device. It can be accessed via a mobile device, tablet or desktop. To maximise your experience whilst using the application it is recommended to use either a tablet or mobile device, however.

Notes

  • EyeLearn is a free-to-use service that aims at making learning a langugae fun and interactive.
  • Maximise the benefits of EyeLearn by using it in a classroom situation.
